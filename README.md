# Cisco ASA Firewall Lab

## Overview
This project demonstrates the configuration of a Cisco ASA firewall, switch, router, DC, chat server, MAIL in a personal lab environment. 

## Objectives
- Configuration of a network with router, switch, vlans and DMZ with focus on CISCO ASA 5508-X.
- Creating Users and GPO

## Lab Topology
- Cisco ASA (routed mode)
- Inside network: 12.164.20.0/24 vlan 20
                  12.164.21.0/24 vlan 30
                  12.164.23.0/24 vlan 40
- DMZ network: 12.164.22.0/24
- Outside network: 192.168.2.0/24

## Tools & Technologies
- Vmware
- Cisco ASA 5508-X
- ASDM

- Windows Server
- Active Directory
